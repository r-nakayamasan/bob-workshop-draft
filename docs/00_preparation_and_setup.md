# 準備と設定

チュートリアルに必要なリソースを準備し、IBM Bobの基本操作を理解します。

### ゴール
- IBM Bobを起動した状態にする
- チュートリアルで使用するデータを手元に用意する
- IBM Bobの画面構成と基本操作を理解する
- チュートリアル中に迷わない状態をつくる


## 環境準備

### 1. チュートリアルデータの取得

#### 1-1. GitHubリポジトリへアクセス

Webブラウザを開き、以下のいずれかの方法でリポジトリにアクセスします：

**方法1: 直接アクセス**
- URL: https://github.com/r-nakayamasan/Bob-workshop

**方法2: 検索**
1. Webブラウザを開く
2. 「github r-nakayamasan bob」で検索
3. 「r-nakayamasan/Bob-workshop」をクリック

#### 1-2. データのダウンロード

1. **「Code」ボタンをクリック**して展開
2. **「Download ZIP」をクリック**

「Bob-workshop-main.zip」という名前のファイルがダウンロードされます。

#### 1-3. ファイルの展開

1. ダウンロードフォルダに移動
2. 「Bob-workshop-main.zip」を右クリック
3. 「すべてのファイルを展開」（Windows）または「展開」（Mac）をクリック

**注意:** .zipファイルではなく、展開後のフォルダを使用します。

#### 1-4. 作業ディレクトリへ移動

展開後の「Bob-workshop-main」フォルダを、作業しやすい場所（デスクトップなど）にコピーします。

**手順:**
1. 「Bob-workshop-main」フォルダをコピー
2. デスクトップ（または任意の作業ディレクトリ）に移動
3. ペースト

### 2. ファイル構成の確認

ダウンロードしたフォルダには以下のファイルが含まれています：

| ファイル名 | 説明 |
|-----------|------|
| **Bob.svg** | Bobのキャラクター画像。FAQサイトに挿入します |
| **qa.md** | FAQ質問集（英語）。日本語に翻訳してサイトに表示します |
| **PROMPT.md** | LAB1用のプロンプト集 |
| **PROMPT2.md** | LAB2用のプロンプト例 |
| **.bobignore** | Bobに読み取らせないファイルを指定 |

**注意:** PROMPT.mdはチャット画面へコピー&ペーストして使用するため、.bobignoreに記載されています。

## IBM Bobの設定

### 3. IBM Bobの起動

デスクトップまたはアプリケーションフォルダから、IBM Bobを起動します。

### 4. プロジェクトフォルダを開く

#### 4-1. フォルダーを開く

1. IBM Bob左上の「ファイル」メニューをクリック
2. 「フォルダーを開く」を選択

#### 4-2. フォルダーを選択

1. 先ほどコピーした「Bob-workshop-main」フォルダに移動
2. フォルダをダブルクリックして中に入る
3. 「フォルダーの選択」をクリック

#### 4-3. ファイルの確認

画面左側のEXPLORERに「Bob-workshop-main」が表示され、以下のファイルが確認できます：

- Bob.svg
- qa.md
- PROMPT.md
- PROMPT2.md
- .bobignore

各ファイルをクリックして内容を確認してみましょう。確認後は、タブ右上の「×」ボタンで閉じます。
